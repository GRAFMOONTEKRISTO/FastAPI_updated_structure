2.  Запуск программы
uvicorn - веб-сервер, который обрабатывает запросы
В терминате прописываем - uvicorn [название главного файла]: [название переменной, которое создает приложение] --reload(автоматическая перезагрузка,если что-то поменялось в коде)
Пример - uvicorn main_application:app --reload
Пример SQL-запроса

Скачиваем библиотеку для моделей - sqlalchemy
Скачиваем библиотеку для миграций - alembic
Скачиваем конектор - psycopg2
Пример (pip install sqlalchemy alembic psycopg2)

Работа с миграциями
1. alembic init migrations - сначала иницилизируем
2. alembic revision --autogenerate -m "Назавание миграции" - создаем ревизию
3. alembic upgrade [код до какой ревизии хочу откатиться]
Удаление талиц в БД - DROP TABLE [НАЗВАНИЕ] CASCADE;
